VAL - Virginia Legislation

Visit website
Github Repository

This is a Django app hosted on Python Anywhere. It collects the information hosted by Virginia's Legislative Information System (LIS), stores it in a database to backup the information and reduce calls to the LIS webpage, and displays the information in a different format. Future planned functionality includes bill tracking, notes, and team-based collaborative tracking and commenting.

The Sign Site

Github Repository

I wrote this to test working with a form to collect and store data using Heroku. It will ask you for your star sign, your personality type, and a random number, then it will show you how you compare to others who have completed the same form.
